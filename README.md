# Spotify Clone 🎵

This is a simple **Spotify Clone** built using **HTML and CSS**.
The project recreates the basic layout and design of the Spotify web player to practice frontend development and improve UI design skills.

## 🚀 Features

* Spotify-inspired user interface
* Responsive layout design
* Sidebar navigation menu
* Music playlist section
* Styled music player footer
* Clean and modern design using CSS

## 🛠️ Built With

* **HTML5** – For structuring the webpage
* **CSS3** – For styling and layout design

## 📂 Project Purpose

This project was created as a **practice project** to improve my understanding of:

* HTML page structure
* CSS styling
* Layout design
* Frontend development basics

## 📸 Preview

You can open the project locally to see the Spotify UI clone in action.

## ▶️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser

## 📚 What I Learned

* Creating layouts using HTML and CSS
* Styling components with CSS
* Building a real-world UI clone
